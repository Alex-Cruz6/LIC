Funciones Math



Función abs()

La función abs(), devuelve el valor abscoluto de un número, para comprender mejor si el usuario ingresa "-1", esta función devolvera como resultado "1".



Función round()

La función round(), redonh2dea un número al número entero superior si la parte decimal es mayor a 0.5 y devuelve un número entero inferior si en la parte decimal es menor a 0.5.



Función ceil()

Esta función, redondea un número al entero superior y si el número ingresado es un entero la función no ejecuta.



Función floor()

Esta función es la contraria a la función ceil(), ya que aproxima el número ingresado al entero inferior y si ingresa un número entero la función no se ejecuta.



Función exp()

La función exp(), es usada para elevar la constante de Euler a un argumento ingresado por el usuario.
Ejemplo: x = exp(2);
En este ejemplo se eleva la constante de Euler a 2.



Función log()

Esta función retorna el logaritmo Natural de un número, su base es Euler(e).
La función log() es equivalente a ln(x).



Función random()

La función random(), retorna un número aleatorio dentro de un rango establecido por el usuario.



Métodos


Método push()

Añade elementos al final de un de un array y devuelve la nueva longitud del array.


Método pop()

Este elemento elimina el ultimo elemento del array y cambia la longitud del mismo.



Función Matricial


Función matricial reverse()

Esta función invierte el orden de los elementos de un array, es decir, el ultimo elemento pasa a ser el primero y así sucesivamente hasta que el primero es el último.
